
#Auto-electron-workspace

##Dependence
 
> node.js

> typescript


##Process (과정)
---

>###1. ```# npm install -g gulp```

>###2. ```# npm install -g electron-prebuilt```

>###3. ```# npm install -g typescript```

>###4. ```# git clone https://github.com/YuJM/auto-electron-workspace.git```

>###5. ```# cd auto-electron-workspace```

>###6. ```# npm install```

##and

###7, ```# gulp ```
